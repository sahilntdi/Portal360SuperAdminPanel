## ‚ö° Quick Tips

1. **Icons are case-sensitive**: `UserCheck` ‚â† `usercheck`
2. **Slugs auto-generate**: "My Article Title" ‚Üí "my-article-title"
3. **Step numbers must be unique**: Can't have two Step 1s
4. **Order = Display position**: 1 appears first, 2 appears second
5. **Ratings scale**: 1 (poor) to 5 (excellent)
6. **Read time examples**: "5 min", "10 min read", "15 mins"

## üîß Common Actions

### Add New Item
1. Click "+" (PlusCircle icon) or Add button
2. Fill in **required fields** (marked with *)
3. Click "Save" or "Add"

### Edit Item
1. Find the item in the table/list
2. Click Edit or Pencil icon
3. Modify fields
4. Click "Save" or "Update"

### Delete Item
1. Find the item
2. Click Delete or Trash icon
3. Confirm deletion

### Upload Image
1. Click "Choose File" or upload area
2. Select image from computer
3. Wait for preview to load
4. Remove with X button if needed

## ‚ùå Common Mistakes to Avoid

| Mistake | Problem | Solution |
|---------|---------|----------|
| Icon name wrong | Icon won't display | Check https://lucide.dev/ |
| Image too large | Upload fails | Compress below 2MB |
| Duplicate step # | Can't save step | Use unique numbers |
| Empty required field | Won't save | Fill all * fields |
| File format wrong | Upload fails | Use PNG/JPG/WebP |
| Spaces in slug | Bad URL | Use "my-article" not "my article" |
